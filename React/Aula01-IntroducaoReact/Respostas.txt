2º Diferença de biblioteca e Framework:

A diferença entre a biblioteca e o Framework é o controle que ambas fornece, ao usar uma biblioteca você possuo mais controle das aplicações, agora se tratando do Framework, o Framework se encarregado do uso das aplicaçõe do código.

4º hooks:

Hooks serve como um armezanamento lógico em que podemos utilizar ou reutilizar nos componentes, o Hooks permite o uso de vários recursos simples nas funções.

5º componentes

Os componente no React é como funções no Java Script, no React há dois tipos de componentes, funcional e de classes, Um componente é um bloco de código reutilizável e independente, que divide a interface do usuário em partes menores, em vez de criar tudo de uma vez só em um único arquivo, os componentes permite divide em seções.
